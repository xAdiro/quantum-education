Nieskończona studnia

n = 1

Rozważmy przypadek, w którym cząstka porusza się między dwoma ścianami o nieskończonym potencjale.

Ściany ustawiami w punktach \(-\frac{a}{2}\) i \(\frac{a}{2}\)

W równaniu Schrodingera $$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2}+V(x)\psi(x)=E\psi(x)$$ dla obszarów 1 i 3 \(V=0\), co dla \(\psi\ne0\) daje wniosek, że \(\frac{d^2\psi}{dx^2} \) jest nieskończone, co nie ma fizycznego sensu.

Zatem: $$\psi_1=\psi_3=0$$

Z kolei dla obszaru drugiego równanie ma postać: \(\frac{d^2\psi}{dx^2} + k^2\psi(x) = 0\),
gdzie: \(k=\frac{1}{\hbar}\sqrt{2mE}\)

Ogólne rozwiązanie tego równania zapisujemy jako: $$\psi = Asin(kx) + Bcos(kx)$$

Wymagamy, aby \(\psi\) było ciągłe pomiędzy obszarami, więc oczekujemy że \(a\) musi być całkowitą
wielokrotnością połowy długości fali co możemy wyrazić jako:

\(a=n\frac{\pi}{k}\),   gdzie   n = 1, 2, 3, 4...

Zatem:

\(\psi=A\sin{kx}\)   dla   n = 2, 4, 6...

\(\psi=B\cos{kx}\)   dla   n = 1, 3, 5...

Gęstośc prawdopobieństwa znalezienia obliczamy jako \(|\psi|^2\)

Korzystając z zależności \(k=\frac{1}{\hbar}\sqrt{2mE}\) i \(ka = n\pi\) możemy obliczyć energię cząstki jako

\(E = \frac{\pi^2\hbar^2}{2ma^2}n^2 \),   n = 1, 2, 3, 4...

Widać więc, że cząstka może przyjmować tylko określone wartości energii.
